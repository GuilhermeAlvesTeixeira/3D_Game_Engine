# 3D Game Engine (Study Project)

This project is a simple game engine built using OpenGL as a way to study graphics programming, engine architecture, and low-level rendering concepts.

## Purpose

The goal of this project is educational. It focuses on understanding:
- Rendering pipeline
- Window and input handling
- Basic engine structure
- Real-time graphics concepts

## Features

- OpenGL context setup
- Basic rendering loop
- Input handling
- Modular project structure

## Technologies

- C++
- OpenGL
- GLFW
- GLAD
- CMake

## Build

```bash
mkdir build
cd build
cmake ..
make
```

## Run 
```bash
./app
```

## Notes
This is not a production-ready engine. It is a learning project and will evolve over time.
